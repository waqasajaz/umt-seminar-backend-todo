## Requirements

### Backend

- nodejs v18
- mysql 8.0.35

# adonis requirements

- https://docs.adonisjs.com/guides/database/introduction

### steps

- create .env file and update environments variables according to requirements
  ```bash
  cp .env.example .env
  ```
- run migration
  ```bash
  node ace migration:run
  ```
- start dev application
  ```bash
  npm run dev
  ```
